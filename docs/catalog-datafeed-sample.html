XML 파일로 연동하기

이 가이드는 카탈로그 소재 정보 제공을 XML 파일로 연동하는 가이드를 제공합니다.

예시

<?xml version="1.0"?>
<rss xmlns:g="http://base.google.com/ns/1.0" version="2.0">
<channel>
    <title>My Deal Shop Products</title>
    <link>https://www.mydealsshop.foo</link>
    <description>Product Feed for Kakao</description>


        <item>
            <g:item_group_id>SKU-123123</g:item_group_id>
            <g:google_product_category>Toys &amp; Games > Toys > Executive Toys > Magnet Toys</g:google_product_category>
            <g:id>SKU-123123-RED</g:id>
            <g:title>WidgetThing</title>
            <g:description>This product is the product you need to do the thing</description>
            <g:link>https://www.mydealsshop.foo/products/widgetthing</link>
            <g:image_link>https://cdn.mycdn.foo/files/123123123.jpg</g:image_link>

            <additional_image_link>https://cdn.mycdn.foo/files/123123123_image2.jpg</additional_image_link>
            <additional_image_link>https://cdn.mycdn.foo/files/123123123_image3.jpg</additional_image_link>

            <g:brand>AcmeCo</g:brand>
            <g:condition>New</g:condition>

            <g:availability>in stock</g:availability>

            <g:price>19.99 USD</g:price>
            <g:sale_price>9.99 USD</g:sale_price>
        </item>

        <item>
            <g:item_group_id>SKU-123123</g:item_group_id>
            <g:google_product_category>Toys &amp; Games > Toys > Executive Toys > Magnet Toys</g:google_product_category>
            <g:id>SKU-123123-GREEN</g:id>
            <g:title>WidgetThing</title>
            <g:description>This product is the product you need to do the thing</description>
            <g:link>https://www.mydealsshop.foo/products/widgetthing</link>
            <g:image_link>https://cdn.mycdn.foo/files/123123123.jpg</g:image_link>

            <additional_image_link>https://cdn.mycdn.foo/files/123123123_image2.jpg</additional_image_link>
            <additional_image_link>https://cdn.mycdn.foo/files/123123123_image3.jpg</additional_image_link>

            <g:brand>AcmeCo</g:brand>
            <g:condition>New</g:condition>

            <g:availability>in stock</g:availability>

            <g:price>19.99 USD</g:price>
            <g:sale_price>9.99 USD</g:sale_price>
        </item>

</channel>
</rss>

제공하는 상품 속성

속성과 타입설명

id

Type: string

필수

Max character limit: 100

상품 고유 ID 이다. pixel&SDK 의 content ID 와 일치해야 한다.

Example: Kakao_tshirt_001

title

Type: string

필수

Max character limit: 225

상품 타이틀이다.

Example: Blue Kakao T-Shirt (Unisex)

description

type: string

선택

Max character limit: 500

상품 설명이다.

Example: A vibrant crewneck for all shapes and sizes. Made from 100% cotton.

availability

Type: string

선택

상품 가용성이다. 아래의 값 중에 하나를 따라야한다. (제약으로 막고있지는 않음)

  • in stock - 즉시 배송 가능한 상품

  • available for order - 배송이 1-2 주 걸리는 상품

  • out of stock - 재고가 없는 상품

  • discontinued - 더 이상 판매하지 않는 상품

Example: in stock

condition

Type: string

선택

상품에 대한 상태이다.

Supported values: new, refurbished, used. (제약으로 막고있지는 않음)

price

type: string

선택

상품 가격과 통화이다. 상품과 갸격 사이에 공백을 두는 3-digit ISO currency code (ISO 4217 standards) 를 따른다. 소수점으로 "." 을 사용한다. 소수점 이하 둘째짜리까지 허용한다.

Example: 9.99 USD, 25.00 EUR

link

Type: string

필수

Max characters: 1000

상품을 구입할 수 있는 URL 이다.

Example: https://www.kakao.com/kakao_t_shirt

image_link

Type: string

필수

Max characters: 500

상품 메인 이미지 URL 이다.

Example: https://www.kakao.com/t_shirt_1.jpg

brand

Type: string

선택

Max characters: 50

브랜드 이름, manufacturer part number (MPN), Global Trade Item Number (GTIN) 중 하나를 선택하여 기입한다. GTIN 의 경우엔 상품의 UPC, EAN, JAN, ISBN 중 하나를 기입한다.

Example: Kakao

additional_image_link

Type: string

선택

Max characters: 500

상품 추가 이미지 URL 이다. 최대 20개까지 가능하다.

Example: https://www.kakao.com/t_shirt_2.jpg,https://www.kakao.com/t_shirt_3.jpg

age_group

Type: string

선택

상품을 사용하는 연령 그룹이다.

지원하는 값: adult, all ages, teen, kids, toddler, infant, newborn

gender

Type: string

선택

상품을 사용하는 성별이다.

지원하는 값: female, male, unisex. (제약으로 막고 있지는 않음)

item_group_id

Type: string

선택

Max characters: 50

상품 그룹 ID 이다. 상품 그룹 ID 가 동일한 상품들은 동일한 상품 그룹으로 묶인다.

Example: Kakao1234_shirts

google_product_category

Type: string

선택

상품에 대한 Google product category (GPC) 값이다. 상품 분류 체계 경로나 ID 를 입력 가능하다.

Example: Apparel & Accessories > Clothing > Shirts & Tops or 212

sale_price

Type: string

선택

할인 가격과 통화이다. 상품과 갸격 사이에 공백을 두는 3-digit ISO currency code (ISO 4217 standards) 를 따른다. 소수점으로 "." 을 사용한다. 소수점 이하 둘째짜리까지 허용한다.

Example: 7.99 USD
