Track ID 발급받기

카카오계정에서는 카카오SDK에서 사용하기 위한 고유한 식별값(Track ID)을 발급할 수 있습니다.
카카오 픽셀을 이용하기 위해서는, 먼저 고유한 식별값이 필요합니다.

https://moment.kakao.com 에서 회원 가입 후,
https://moment.kakao.com/mypixel 페이지에서 Track ID 발급 단계를 진행할 수 있습니다.

Track ID 발급이 완료된 후, 다음 단계의 안내에 따라 카카오 픽셀을 웹사이트에 설치해주세요.
