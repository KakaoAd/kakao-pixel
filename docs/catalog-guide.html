카탈로그 상세 설정

이 가이드는 카카오 픽셀을 통한 카탈로그 소재 설정에 필요한 상세 가이드를 제공합니다.

OpenGraph

  • 픽셀&SDK를 통한 상품정보 수집은 픽셀에서 발생하는 '컨텐츠/상품 조회' 이벤트를 통해 이뤄집니다. 이벤트를 통해 상품이 정상적으로 사용되기 위해서는 아래 제공된 OpenGraph 가이드에 맞춰 상품 페이지가 설정되어 있어야 합니다.
OpenGraph 예
<meta property="og:title" content="Kakao Product">
<meta property="og:description" content="Kakao Product Description">
<meta property="og:url" content="https://www.daum.net">
<meta property="og:image" content="https://t1.daumcdn.net/daumtop_chanel/op/20200723055344399.png">
<meta property="product:brand" content="KAKAO">
<meta property="product:availability" content="in stock">
<meta property="product:condition" content="new">
<meta property="product:price:amount" content="10000">
<meta property="product:price:currency" content="KRW">
