픽셀 기본 사항 안내

카카오 픽셀에 관련된 기본 사항을 안내합니다.

기본 수집 항목

  • 카카오 픽셀을 통해 수집하는 항목은 다음과 같습니다.
    • ad.daum.net에서 발급한 쿠키에 포함한 비식별화된 식별값
    • 이벤트 종류 및 태그
    • 픽셀이 작동한 웹페이지 URL
    • 픽셀이 작동한 웹페이지로 이동하기 전 URL
    • IP 주소
    • 브라우저 User Agent

데이터 전송 확인 방법

  • 픽셀&SDK 관리페이지 내에서 이벤트별 목록 옆에 아래와 같이 "수집중" 이라고 표시되면 정상 데이터가 수집중입니다.
  • "수집중" 녹색 텍스트를 클릭하시면 아래와 같이 최근 수집된 100건의 이벤트 목록을 확인할 수 있습니다.

태그 추가

  • 픽셀&SDK 관리페이지 내에서 "+태그 발급하기" 버튼을 눌러 이벤트 하위에 상세한 추가 설정을 할 수 있습니다.
  • 태그는 카카오모먼트 광고그룹 내에서 '맞춤타겟 > 내 데이터 > 픽셀&SDK' 항목으로 타겟팅에 사용할 수 있습니다.
  • 픽셀로 보내는 이벤트에 '태그'를 추가하여 보내는 경우, 이벤트의 타겟모수와 이벤트 하위 태그의 타겟모수에도 함께 포함합니다.
  • "+태그 발급하기"로 추가하지 않은 '태그'를 스크립트 내에 추가하여 전송하는 경우, 이벤트의 타겟모수에만 모수로 포함합니다.
